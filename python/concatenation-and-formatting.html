Python: Concatenation and String Formatting

Learning all of the tricks to varibles and strings.

Introduction to Concatenation

What is concatenation? By simple definition it means "to link together". Relating to Python, this particular method allows us to combine any number of things like lists, tuples, strings, or dictionaries together into one piece of code. Concatenating isn't always adding-- it can be related to any mathematical equations in any code you're working on. Our first example is with strings.

>>> var1 = "exa"
>>> var2 = "mple"
>>> var1 + var2
'example'

This is a great start, but can become messy and inefficient: in order to call it again, you need to rewrite the var1 + var2. Although this isn't terribly extensive, this can become an issue in longer, more realistic examples. Here's where concatenation comes in. By creating a new variable and assigning the operation directly to it, we can call the new variable, and it will print the addition and simultaneously change the values in the two original strings.

>>> var3 = var1 + var2
>>> var3
'example'

Here's another example with multiplication:

>>> first_value = "red"
>>> first_value * 3
'redredred'

Since the original string didn't include a space after the word 'red', the output is one unbroken word, and it can easily be changed to fit what you're going for. This same principal demonstrated here can be applied in the making of a new varible, then putting the operation inside it.

String Configuring

In the example above, the output came out as one string. Even if we included the extra space at the end of the original string, the end result would still end up as a single string of three words. What we want to achieve for the sake of this experiment is to come up with three separate strings of our initial multiplied string. This can be done by using the split() method.

>>> foo = "red"
>>> foo.split() * 3
['red', 'red', 'red']

It is important to rememeber that this method puts the value into a list. While there are many different complicated ways in Python to seperate the strings, this is generally the simplest way to do it.

format() Function

How about a string that changes depending on certain values inside a varible? The format() function is probably the best way to demonstrate this idea. Let's take a look at it in action.

>>> age = 35
>>> name = "Jerry"
>>> string = "Hello, my name is {} and I'm {} years old".format(name, age)
>>> string
"Hello, my name is Jerry and I'm 35 years old"

As you can see, the two brackets are very important in this example. They act as a placeholder for our changing value. Then, inside the format() we can insert whatever variable names we want. This ends up with our variables being included in the final output of the string that is printed out. You can also achieve this without the brackets and with a percent sign and s - %s.

In future demonstrations, we will learn more advanced examples with concatenation and string formatting, going a bit further with this method's extensive uses.

Python: Concatenation and String Formatting

Introduction to Concatenation

What is concatenation? It simply means to "link together". This means in Python you can combine any number of things like lists, tuples, strings, or dictionaries. Now concatenating isn't always adding. You can apply any mathematical equations to anything you're working on. Here's our first example with strings.

>>> var1 = "exa"
>>> var2 = "mple"
>>> var1 + var2
'example'

This is cool but if you want to call this again, you have to write this again: var1 + var2. Instead there's a better way. You can make a new varible and assign the opperation to it. Now if we call the new varible, it will print what we added but we can also change the values in the two original strings.

>>> var3 = var1 + var2
>>> var3
'example'

Now lets try out a new example with multiplication.

>>> first_value = "red"
>>> first_value * 3
'redredred'

The output is all one word and that's because the original string didn't have a space in it. You can apply the same principal here also with making a new varible and putting the opperation inside.

String Configureing

Now in the example above, our proplem was making all of the words separate. Yes we could of just put a space in the string so it would print looking like they're separate, but it's still one string. What we want to achive it to get three separate strings. We can do this using the split() method.

>>> foo = "red"
>>> foo.split() * 3
['red', 'red', 'red']

One thing to note, this method puts the value into a list. This is proberly the best way to keep strings separate.

format() Function

What if you want a string that would change depending on certain values in varibles. You can do this with the format() function. Lets see it in action.

>>> age = 35
>>> name = "Jerry"
>>> string = "Hello, my name is {} and I'm {} years old".format(name, age)
>>> string
"Hello, my name is Jerry and I'm 35 years old"

The two brackets are very important in this example. They act as a placeholder and then inside the format() we put the varible names so it puts whatever the value is in the varible and prints it with the string. You can also achive this without the brackets and with a percent sign and s - %s.

We will futher learn more advanced examples with concatenation and string formatting.

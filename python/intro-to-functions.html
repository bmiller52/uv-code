Python: Intro to Functions

Learning the concept of functions

Introduction

What are functions? In simple terms, they are blocks of code that can be reused however many times you want to call on them. Python already comes with predefined functions like print() and raw_input(). However, as coders we can still make our own functions to make the code cleaner and more compact. By typing in def (which abbreviates define), the function name, followed by an opening and closing parentheses and a colon, you've created a function. From there, you can fill it with whatever you need to be inside that function.

def profile_setup():
  question1 = raw_input('Hello, what is your name? ')
  question2 = raw_input('How old are you? ')
  question3 = raw_input('What\'s your occupation? ')

  print()   print('Name: ' + question1)
  print('Age: ' + question2)
  print('Occupation: ' + question3)

While this is a correctly formatted function, nothing will happen if this code is run in a script or the Python console. The reason for this is that we only defined the function, but never called it. This can easily be fixed by typing in the function name and parenthesis.

profile_setup()

By calling it, the code set up inside the function will run, prompting the three questions, then outputting the responses. This is extremely useful, since it allows you to reuse the same pieces of code however many times needed. Now, when needing a large segment of code multiple times to be run throughout the rest of the code, that code can be put inside a function, and called on seperately each time. Several other useful benefits to functions is how they eliminate higher chances of syntax errors and repetitively convoluted code.

Going Further with Functions

What if the user wants to enter in some other code including the function's code, before actually running the function? Many coders will do this to determine the output of the function. Here's a basic example of this.

def do_math(x, y):
  calculation = x + y
  return calculation

num1 = int(raw_input('Pick any number '))
num2 = int(raw_input('Pick another number '))

do_math(num1, num2)

This script allows the user to determine two numbers that the function will add. What is inside the parenthesis when defining the function is arbitrary: they are essentially variables. Notice the int function (a pre-defined Python function) that is outside of the raw_input. Without this, the program wouldn't run. In simple terms, it parses whatever the user inputs to be read as an integer. Otherwise, the console would look at it as a string and would concatenate it, which isn't what we want in this example. This way, though, it adds the two numbers by recognizing them as integers and through this function method.

Conclusion

Functions are very commonly used by the most professional Python coders, but they aren't a difficult concept to grasp, which is why they are so universally used. With this in your pocket, it will be much easier to work with Python, especially for coding DRY (Don't Repeat Yourself). In games, programs, and practically any other use for Python, functions are essential to keeping your code clean and DRY.

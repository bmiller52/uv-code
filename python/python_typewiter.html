Python: Typewriter Effect

Learn how to use the typewriter effect in Python and what requirements goes along with
it. We will go in-depth with what code you need and why it works.

The Code

I'll just start off with giving you the code, then explain why this works and what else you can do with it.

from random import uniform
from time import sleep
import sys

example = 'This is just a test so you can see how this trick works. Pretty cool huh?'

for x in example:
  print(x, end = '')
  sys.stdout.flush()
  sleep(uniform(0.01, .3))

print(' ')

Requirements

Python 3 or higher

Why this works

In the first 3 lines of code, we have some basic imports. All three are built in modules so you don't have to download anything. The next line is a varible and we're storing a string in it. This will be are example text to print. Now comes the typewriter effect. We will start with a for loop and then declare another varible. In this case, we named it x. Remember this is just another varible, so it's arbitrary. Now we call example which we defined earlier. Now we indent the next line so it's inside the for loop. Next we print x. So what the code is saying is that for every character in example, which is a string, we print one character. Since we're using a for loop, this will go round and round till it prints the entire varible. Now we add sys.stdout.flush(). This basically tells the terminal to "flush" the buffer. Now is the fun part. The next line is a sleep function. This will be combining the sleep and random modules we imported. Normally when we use the sleep function, we just put a number (representing seconds) to delay the process. But if we add the uniform function after the sleep, this allows us to pick a random float. This will complete the effect and will print out each character one by one randomly to give it a great effect. Go ahead. Play around with the floats and see the diffrent delays you get.

IMPORTANT! Note: This will only work with Python 3 or higher. This capability is a new feature in the Python 3 release.

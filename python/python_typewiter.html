Python: Typewriter Effect

Learning how to use the typewriter effect in Python and exploring the code in-depth

Introduction: The Code

First, let's begin by demonstrating the finished product of code. From there, the exact function and application of how to incorperate this technique into your full code can be explained.

from random import uniform
from time import sleep
import sys

example = 'This is just a test string of words so you can see how this trick works. Pretty cool, huh?'

for x in example:
  print(x, end = '')
  sys.stdout.flush()
  sleep(uniform(0.01, .3))

print(' ')

How it works

In the first three lines of code, we have several basic imports. All three are built-in modules, so no extra downloads are necessary. The next line is a basic varible used to store a string of words in it. By the end of the code, this will be our example text to demonstrate the typewriter text effect.

With the basic items out of the way, we can get to work on the actual effect. We will start with a for loop, then declare another varible. In this case, we named it x. Remember, this is simply another variable, so it's completely arbitrary. Now we can call on our variable example, which we defined earlier. Here's where indenting is important to remember: the next few lines should be inside the for loop, so we make sure to indent it so that it fits nicely in there. Now that we're inside the loop, print x. What we have so far translates that for every character in example, which is a string, a character is printed. Since we're using a for loop, this will go round and round until it gets the job done: printing the entire varible. The next addition is adding in sys.stdout.flush(). This is important to incorperate, as it tells the terminal to "flush" the buffer .

Here comes the fun part. The next line needed is using the sleep function. This will be combining the sleep and random modules we have already imported. Normally when we use the sleep function, we can simply input in a number (representing seconds) to delay the process. However, by adding the uniform function after sleep, this allows us to pick a random float rather than the typical system Python uses. This will complete the effect by printing out each character one by one in random order. Of course, the floats aren't stuck to this one function: many different delays and patterns can be experimented with to achieve the effect that looks right for the type of formatting you're looking for. Go ahead and play around with it!

Requirements

IMPORTANT NOTE: This will only work with Python 3 or higher. The capability is a new feature in the Python 3 release and will not function in any lower versions of Python.

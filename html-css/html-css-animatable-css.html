HTML/CSS: Animatable CSS

Learning the many unique properties of CSS

Introduction: Can CSS be Animatable?

While the concept seems difficult, the execution can be done very easily. In most cases, Javascript is the most common method used for animations in this form of coding. However, by incorperating CSS into our code instead, we can find a fast and understandable approach. A great example is a simple box that fades into different colors. Let's take a look at the code.

HTML

<div id="empty-box"> ... </div>

CSS

#empty-box {
  width: 300px;
  height: 200px;
  background: red;
  animation: test 5s infinite;
}

@keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

As you can see in the HTML, we merely created a box using a div tag. Now all we need to do is edit the CSS. First, we set some basic height, width, and color properties for our example box. This gives us a clean, beginner box to work with. The next line has an animation property. Using this property is a key to the code--it allows for the colors to fade and blend together smoothly. Inside the animation property, you need to give it a variable name (in our example case: 'test'), how long the fade should be ('5s'), and determine the length of the cycle ('infinite'). For the sake of the example, we don't want the cycle to stop so we use infinite. In the next style, we use @keyframes, which is also part of the CSS3 transitions. Now it's time to call the varible we set earlier, test, to target that specific part of code. Finally, set a start and an end for the colors in the box with the background-color property. By setting these colors, we can let the animation property do it's magic and enjoy the finished product.

Unfortunately, this initial product of code is not supported in Chrome, Safari, or Opera. However, by doing a small bit of extra work in your CSS, this can be easily fixed by adding the following piece of code to your #empty-box style.

-webkit-animation: test 5s infinite;

Finally, add in one more piece of style.

@-webkit-keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

While it isn't supported by Internet Explorer 9 and earlier versions, this CSS trick is completely compatible with practically all other browsers now. In addition, it is not the limt to animations in CSS. Techniques such as inflatable and deflatable elements, rotating elements, and etcetera are also available and easily accessable in CSS. Now that we're finished, here's the complete code for the color-changing box:

#empty-box {
  width: 300px;
  height: 200px;
  background: red;
  animation: test 5s infinite;
  -webkit-animation: test 5s infinite;
}

@keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

@-webkit-keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

Note: This will not work in Internet Explorer 9 and earlier versions.

Difficulty: Intermediate

HTML/CSS: Animatable CSS

Learn how CSS can be used in many diffrent ways.

Can CSS be Animatable?

Why yes it can. Normally you would think you use jQuerry for animations, and in most cases you do. But here's a fun trick to try out using CSS. It's a simple box that fades colors. Let's see the code.

HTML

<div id="empty-box"> </div>

CSS

#empty-box {
  width: 300px;
  height: 200px;
  background: red;
  animation: test 5s infinite;
}

@keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

Now as you see, we just created a box using a div tag. Then in our CSS, we set some basic height, width, and color properties for our example box. The next line has this animation propertie. Now what's that? This is how we fade the colors together. We first give it a variable name, then tell how long the fade should be, then if you ever want to stop it. In out case we don't want it to stop so we use infinite. In the next style, we use @keyframes, which is also part of the CSS3 transitions. Then we call the varible we set eralier, test. Now we just need to set a start and an end for the colors. With setting these colors, the animation property will do it's magic. Now lets see it work.

This is great and all but this is not supported in Chrome, Safari, or Opera. Is there a way to do this in these other browsers you ask? Yes! Of course. In your #empty-box style, add this piece of code.

-webkit-animation: test 5s infinite;

Then add a this new style.

@-webkit-keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

Great, now it works in most browsers. I say most because this is not supported in Internet Explorer 9 and earlier versions. This is not the limt to animations in CSS. You can do many other things like changing sizes of elements, roatating elements, ect. Now here's the complete code for the CSS.

#empty-box {
  width: 300px;
  height: 200px;
  background: red;
  animation: test 5s infinite;
  -webkit-animation: test 5s infinite;
}

@keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

@-webkit-keyframes test {
  from {background-color: red;}
  to {background-color: blue;}
}

Note: This will not work in Internet Explorer 9 and earlier versions.

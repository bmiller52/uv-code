Bootstrap: Intro to Bootstrap

Uncovering the vary basics of Bootstrap

Introduction

Lets start with what Bootstrap is. It's simply a framework that helps your workflow for building websites. It includes HTML, CSS, and Javascript. The main diffrence comes in the CSS. The CSS file has more then 9,000 lines of code. Bootstrap also has a full-flegded documentation. It's extremely detailed and has every possible thing you can imagine. Fun fact: Bootstrap was made by Twitter and companies like Spotify and Lyft use this framework for their websites. Now what's some pros and cons?

Pros

  • Easy to use documentation
  • Vary fast workflow
  • Customizable content (obviously)
  • Works in all modern browsers
  • Responsive

Cons

  • Your site can look vary Bootstrapy
  • CSS can be diffucult to customize if you are inexperienced
  • jQuery plugins are limited

What do I mean that your website can look Bootstrapy? Well there comes to be a point where most Bootstrap sites look all the same. You're using the same code from the documentation that everybody else uses. You can shake things up in the CSS but that can take some time. But go ahead and explore in the CSS files. Make things your own.

Other Frameworks

Bootstrap isn't the only good framework out there. There are some other ones that are becoming more popular, like Foundation or Skeleton. They all result in the same problem with websites all looking the same. Try some others out and see if you like them.

Starting with Bootstrap

The set-up page in the documentation should get you started but there are some other things you need to know. First, you may notice some code snippets with links to CSS and Javascript files. Thoes links go to webaddresses which with work but you won't be able to change any files because it's pulling from the web. You can however, download all of the files, then link to the local file so you have full customizable power to your Javascript and CSS. I would recommend using the starter template below the Quick Start. Have fun with your first website!

HTML/CSS: The Link Tag

Understanding the abilities of the link tag

Introduction

We have already discussed how to link to a external file in a pervious tutorial - Linking to External Files. However, the link tag can also be used in several other ways. A common element that nearly all modern websites have today is an icon in the top right corner of a tab (Except for Safari, which takes out all icons by default). The icon not only acts as a strong marketing technique, but also as a link, which we will be focusing on. In order to master this, we need just two attributes - rel and href. Remember that rel is short for relationship and href is the Hyperlink Refrence.

<link rel="icon" href="logo.jpg">

As you can see, it's a fairly simple process. The most important part here is making sure that the path to the image is correct. Depending on where the picture is saved, you may have to go in or out of multiple foldes to have the code run correctly.

Other Uses

The rel attribute, as previously stated, has an extensive amount of uses. It is a key element to HTML for multiple reasons, as seen in the list of possible uses for it below.

  • alternate
  • author
  • dns-prefetch
  • help
  • icon
  • license
  • next
  • pingback
  • preconnect
  • prefetch
  • preload
  • prerender
  • prev
  • search
  • stylesheet

If you use Brackets, the program should come up with this list of suggestions as soon as you type in rel = . Let's take a closer look at a few of them to try to understand the importance of them.Preconnect is used to tell the browser to preemptively connect to the origin of the resource. While this sounds confusing, it really is simple and helpful. All it does is make sure the code for whatever you're linking files to runs before anything else--a type of headstart before the other code so that all of the links will run smoothly once the actual website loads. That can be really useful. Others like next and license provide links to other documents and information . However, the most common ones you will likely use will be icon and sylesheet, as they import things that are essential to every webpage. To see more on the stylesheet value, see Linking to External Files.

The best part is here! Let's start with the rules of thumb. First, every tag has a opening and most of the time has a ending tag. The first tag starts with a less then sign: <, then comes the grater then sign: >. For example, a basic header 1 tag would look like this.

So whatever you want to display on the webpage, you would put inbetween the opening and the closing tags. Lets see an example.

Would output this -->

You see how we put the words we wanted to display inbetween the two tags? Now lets look at the important of the framwork of the code.

Almost every webpage has this piece of code at the very top of their page:

<!DOCTYPE html>

This just tells the browser that it's using HTML5. You only need to use it one time per page. Next you need this:

<html>
...
</html>

We need to put a head tag in now. We will put everything that we don't what to be displayed on the screen in here. So you would put maybe some styles or links in there, we will learn about those later. Example:

<html>
  <head>
  ...
  </head>
</html>

Now we can put the body tag after the head tag. So whatever we want to be displayed, we would put in these body tags. Lets put everthing together now:

<!DOCTYPE html>

<html>
  <head>
  ...
  </head>
  <body>
  ...
  </body>
</html>

Great! Now we have the correct fromat for are web page. Here's a cheat sheet so you can experiment with tags and see what you can print on the screen. http://www.simplehtmlguide.com/cheatsheet.php

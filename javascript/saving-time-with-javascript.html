JS: Saving Time With JavaScript

Maximizing efficiency in workflow

Introduction

In JavaScript, even the most basic coding can be time-consuming and ineffecient. Let's create a simple example website to demonstrate our time-saving technique. Say we have a website with 20+ pages. Most modern websites have a layout and color scheme that stays consistent for each page, and is compatible with the generic elements, such as the logo/title, menu/navigation bar, the body of the content, and the footer. In order to change any given aspect of the website, you'll have to go through each file and change it manually, which can take some time and become tedious. An easy way to eliminate that is by using id's.

Digging Deeper

Instead of changing each file, you can give each list item an id, as seen in the HTML.

<ul>
  <li><a href="index.html" id="home_value"></a></li>
  <li><a href="all-posts.html" id="post_value"></a></li>
  <li><a href="contact.html" id="contact_value"></a></li>
</ul>

Now that we have the HTML correct, we need to incorperate three simple lines of JavaScript.

document.getElementById("home_value").innerHTML = 'Home';
document.getElementById("post_value").innerHTML = 'All Posts';
document.getElementById("contact_value").innerHTML = 'Contact';

The getElementById is used to target a place where we want to insert the text. Inside the parenthesis, we add the id name. To do this by using classes we could have used getElementByClass instead. Now this current script acts as the master control for the website. By simply changing the value in this script, it will carry this order on in every file you have, saving hours of time for websites with a large amount of files.

With a few simple tricks in JavaScript, your website work can become much easier in the long run.
